=====================ppocr 中文-单字符检测========================

	python tools/train.py -c configs/det/ch_PP-OCRv3/ch_PP-OCRv3_det_student.yml

2、测试文件夹下所有图像的检测效果：
	python tools/infer_det.py -c configs/det/ch_PP-OCRv3/ch_PP-OCRv3_det_student.yml -o Global.pretrained_model="./output/ch_PP-OCR_V3_det/best_accuracy"

	python tools/export_model.py -c configs/det/ch_PP-OCRv3/ch_PP-OCRv3_det_student.yml -o Global.pretrained_model="./output/ch_PP-OCR_V3_det/best_accuracy" Global.save_inference_dir="./output/ch_PP-OCR_V3_det/inference/"

4、使用 inference 模型预测：
	python tools/infer/predict_det.py --det_algorithm="DB" --det_model_dir="./output/ch_PP-OCR_V3_det/inference/" --image_dir="./train_data/single_character_data_det-by-yolov5_bunch/eval/" --use_gpu=True --det_limit_side_len 736 --det_limit_type min

	python tools/train.py -c configs/rec/en_PP-OCRv3/en_PP-OCRv3_rec_char.yml

5、# 预测英文结果
	python tools/infer_rec.py -c configs/rec/en_PP-OCRv3/en_PP-OCRv3_rec_char.yml -o Global.pretrained_model="./output/en_PP-OCRv3_rec_char/best_accuracy"

6 、训练模型导出为inference模型：
	python tools/export_model.py -c configs/rec/en_PP-OCRv3/en_PP-OCRv3_rec_char.yml -o Global.pretrained_model="./output/en_PP-OCRv3_rec_char/best_accuracy" Global.save_inference_dir="./output/en_PP-OCRv3_rec_char/inference/"

7、使用 inference 模型预测：
	python tools/infer/predict_rec.py --rec_model_dir="./output/en_PP-OCRv3_rec_char/inference/" --image_dir="./train_data/single_character_data_det-by-yolov5/test/crop_img/" --use_gpu=True --rec_char_dict_path ppocr/utils/en_dict.txt --use_space_char false
